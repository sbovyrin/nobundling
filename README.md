# Зачем нужен bundle manager

## Dev
- import a file (css, json, js)
  - `import './style.css'`
  - `import React from 'react'`
- hot reload
- dev server
- transpilation to old JS
- autoprefix CSS

## Prod
- transpilation to old JS
- autoprefix CSS
- concatenate JS files to one or multiple
- concatenate CSS files to one or multiple
- minify JS and CSS
